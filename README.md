# 🛒 Digiboi v4.0 — বাংলাদেশি POS ও ব্যবসা ব্যবস্থাপনা

## 🚀 Features
- ✅ POS বিক্রয় (স্টক লিমিট সহ)
- ✅ ইনভেন্টরি ব্যবস্থাপনা (QR/Barcode)
- ✅ কাস্টমার প্রোফাইল ও ক্রয় ইতিহাস
- ✅ বাকি কালেকশন (দশমিক সহ সঠিক হিসাব)
- ✅ সরবরাহকারী ব্যবস্থাপনা
- ✅ খরচ ব্যবস্থাপনা (তারিখ ও delete সহ)
- ✅ রিপোর্ট ও বিশ্লেষণ
- ✅ Notification সিস্টেম
- ✅ রিকভারি কোড দিয়ে পাসওয়ার্ড রিসেট
- ✅ Bangladesh UTC+6 timezone
